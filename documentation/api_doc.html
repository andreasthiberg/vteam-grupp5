API

Elsparkcyklar

Introtext.

En elsparkcykel har följande attribut:

Attribut 1
Attribut 2
Attribut 3

Hämta information om alla cyklar

GET /scooters

Resultat:

{
    "data": [
        {
            "id": 1,
            "att1": "?",
            "att2": "?",
            "att3": "?"
        },
        {
            "id": 2,
            "att1": "?",
            "att2": "?",
            "att3": "?"
        }
    ]
}

Hämta information om en cykel

GET /scooters/:scooter_id

Resultat:

{
    "data": [
        {
            "id": 1,
            "att1": "?",
            "att2": "?",
            "att3": "?"
        }
    ]
}

Skicka uppdaterad information om en särskild cykel

PUT /scooters/:scooter_id

Nödvändiga parametrar:

id
authetication?

Möjliga parametrar:

position
status
etc

Skicka ett kommando till en cykel

POST /scooters/command/:scooter_id

Nödvändiga parametrar:

id
authetication?

Möjliga parametrar:

status
command
